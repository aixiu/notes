FlashPlayer提示地区不相容的解决办法

发表于 2018-06-10 分类于 学习笔记 阅读次数： 本文字数： 610 阅读时长 ≈ 2 分钟

MMP的2144

这两天打开Chrome提示FlashPlayer与您的地区不相容，请重新安装FlashPlayer,心里突然一万个草里马，本来FlashPlayer只是一个播放工具，和版本没多大关系，和地区就更没什么关系了，不过为了一些漏洞和安全，才有了升级版本，现在FlashPlayer被国内2144接管后，更是加了地区限制,MMP。
2144版就算是下载离线安装后，还是会出现被限制使用，虽然那个在线包没进行推广等流氓行为 但是只能下29的正式版 也难免以后会不会加料。

解决办法

离线安装方法

提供了去掉地区限制的离线安装版本。注意：一定要离线安装。

AX PP NP 都去掉了地区限制
IE用AX
Chrome系用PP
Firefox或其它用NP

本特别版：
基于国际版最新 30.0.0.154 制作，包含 AX/NP/PP 三种版本
已经屏蔽删除反馈服务和升级服务程序
解除中国大陆地区无法安装国际版 Flash 的限制
安装过程无需断网，卸载过程也无需断网
解除使用国际版 Flash 后提示地区不符无法使用的限制
安装前自动检测系统是否安装，若安装，运行旧版本卸载程序让用户手动卸载

PS：若安装有旧版本，请卸载后注销或重启再安装，确保效果

其他说明（部分引用自闻雷）：

Windows 10 的 Edge 和 IE 的 ActiveX 版 flash 是通过系统补丁的方式推送安装，这样安装的 Flash 插件没有区域限制问题，无须替换。
Google Chrome 自身更新的 PepperFlash 也是有地区限制的，所以——
如果不在系统里安装 PPAPI 版 Flash，而使用 PepperFlash 的话，也要用上述修改好的文件替换；
如果使用系统里安装的 PPAPI 版 Flash，那么建议屏蔽 Chrome 自身更新的 PepperFlash，方法如下：
直接删除 chrome 的 User Data 目录下的 PepperFlash 文件夹
再建立同名占位文件（注意把扩展名删了），设置为只读
注意：ActiveX 版本不能在高于 Windows 7 以上的系统启动安装包
