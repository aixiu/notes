WOW插件中英文对照1.133可用(不冲突)列表

于2019年10月18日第二版收集完成。插件列表REAADME.md文档中有提示用法和注意事项。

使用需知

  • 60Addons-Classic-list 为插件包，下载后，安装目录为你的目录\World of Warcraft\_classic_\Interface\AddOns下

  • Fonts 为字体包，下载后，安装目录为你的目录\World of Warcraft\_classic_\Fonts下

  • Interface-材质包 为材质包配合AbyssUIClassic插件使用（也可以不用这些材质，不影响正常使用插件），下载后，安装目录为你的目录\World of Warcraft\_classic_\Interface\AddOns下

常用设置及宏

  • 反和谐设置方法(反骨头和墓碑和谐)

    • 第一种方法
      目标文件：X:\World of Warcraft\_classic_\WTF\Config.wtf
      打开方式：使用记事本打开
      最后一行添加：
      SET overrideArchive "0"
      然后保存文件，重启游戏就好了。

    • 第二种方法种方法(经坛友提醒也可以通过游戏内直接执行命令的方式设置，已实测可行)
      进入游戏后在聊天窗口直接输入
      /console SET overrideArchive "0"
      然后重新启动游戏。

  • 反语言屏蔽 SET profanityFilter "0" (修改方法同上)

  • 注意：建议选择亡灵的玩家都打开反和谐设置，因为这样在创建人物选择特色时，能够看到完整的每个特色。

  • /run SetCVar("cameraDistanceMaxZoomFactor", 4) 视距宏

  • /console SET targetNearestDistance "50" Tab距离

  • /console SET chatClassColorOverride "0" 职业聊天颜色

  • 打开装备自动比较 /run SetCVar("alwaysCompareItems", "1") 1代表打开 0代表关闭

  • 设置血液效果等级 /console violenceLevel 2 最后一位数字代表暴力等级 默认是2 最大是5 最小是0

  • 隐藏动作条两边的狮鹫 左右各一个

    • /script MainMenuBarLeftEndCap:Hide()
    • /script MainMenuBarRightEndCap:Hide()

  • 友方姓名板显示职业颜色 /console ShowClassColorInFriendlyNameplate 1 0代表关闭 1代表打开

  • 敌方姓名板显示职业颜色 /console ShowClassColorInNameplate 1 0代表关闭 1代表打开

  • 副本重置宏 /run ResetInstances()

  • 目标宏

    • 注：0 = 取消标记，1 = 星，2 = 大饼，3 = 菱形，4 = 三角，5 = 月亮，6 = 正方形，7 = X，8 = 骷髅

字体设置

  • 字体文件夹在哪?
    • 怀旧服：World of Warcraft/classic/Fonts (没有的话自己创建个)
    • 正式服：World of Warcraft/retail/Fonts 怀旧服和正式服是分开的，可以分开设置
  • Fonts里的各个文件分别是什么内容？
    • 正式服Fonts文件夹下自带了6个文件，分别是ARHei、ARIALN、ARKai_C、ARKai_T、FRIZQT__、skurri，后缀都是.ttf
    • 主要三个字体文件如下：
      • ARHei：聊天文字、物品说明、技能说明
      • ARKai_C：战斗伤害数字、头像血条/等级数字、小地图时间数字、背包金钱/物品堆叠数字
      • ARKai_T：登陆界面、游戏界面主字体、任务说明、头像名、物品名、技能名等等等等。。。
      • 基本上除了前两个的范围以外全都是这个文件的字体
  • 如何替换字体？
    • 简单来说，首先你要去找一个.ttf后缀的字体文件，然后按上面的命名方式改名，最后放入Fonts文件夹下替换掉原来的。
